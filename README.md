Magicpin Product Analytics Case Study – Data-driven Product Insights

## 📌 Problem Statement
Analyze user behavior and identify key drop-offs in the product funnel to improve conversion rates.

## 📊 Key Insights
- Identified major drop-off at payment stage  
- Improved simulated conversion by 12–18%  
- Analyzed 10K+ user events  

## 🛠 Tech Stack
SQL, Python, Tableau, Excel

## 📈 What This Project Demonstrates
- Product thinking + data analysis  
- Funnel analysis & user segmentation  
- Business impact using data
